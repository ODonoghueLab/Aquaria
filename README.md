# aquaria-vuerouter

## Project setup
```
npm install
```
----------
### Run in development mode
```
node app.js
npm run serve
```
----------
### Run in production mode

Before running on production mode edit the 'app.js' file to use dist folder instead of public.
```
npm run build
node app.js
```
Load page on localhost:8010

----------
### Lints and fixes files
```
npm run lint
```
