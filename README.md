# aquaria-vuerouter

## Project setup
```
npm install
```
----------
### Run in development mode
```
npm run bundle-legacy
npm run serve
```
----------
### Run in production mode

Before running on production mode edit the 'app.js' file to use dist folder instead of public.
```
npm run bundle-legacy
npm run build
```

----------
### Lints and fixes files
```
npm run lint
```
